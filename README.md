# BMI-Rechner
Eine dynamische Website mit HTML, CSS und Python CGI auf pan.th-brandenburg.de

##Beschreibung
Der Benutzer gibt Name, Gewicht und Größe ein.

Ein Python-CGI-Skript berechnet den BMI und zeigt die Ergebnisseite an.
